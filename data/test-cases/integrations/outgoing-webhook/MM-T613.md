## MM-T613: Disable outgoing webhooks in System Console

---

**Step 1**

Disable outgoing webhooks in System Console\
–––––––––––––––––––––––––

1. System Console ➜ Integrations ➜ Integration Management
2. Set `Enable Outgoing Webhooks` to `false`
3. Save
